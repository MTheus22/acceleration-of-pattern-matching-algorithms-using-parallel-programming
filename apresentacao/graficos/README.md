# Graficos

Pasta para graficos da apresentacao, preferencialmente gerados por scripts.

Regras:

- A fonte primaria dos numeros e o texto atual do TCC.
- Se for necessario consultar o banco, use a coleta canonica da workstation e
  confira o resultado contra `../partes/results.tex`.
- Salvar saidas em SVG quando possivel.
- Cada grafico deve destacar uma unica mensagem.

Graficos provaveis:

- queda de throughput por footprint do automato;
- curva de speedup por threads;
- ganho do layout de saida plana;
- sharding de dicionario contra particionamento de texto;
- composicao 2D contra particionamento de texto;
- end-to-end sobre Enron 8x.
