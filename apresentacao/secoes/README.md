# Secoes da apresentacao

Cada arquivo desta pasta descreve uma secao da defesa com escopo reduzido para
agentes que forem trabalhar em partes especificas do deck ou do roteiro.

Use assim:

1. Leia `../AGENTS.md`, `../DESIGN.md` e `../base-apresentacao.md`.
2. Abra apenas o `secao-XX.md` da parte que sera editada.
3. Consulte `../../partes/*.tex` somente para conferir narrativa ou numero que
   nao esteja claro na base.

Mapa:

| Arquivo | Secao |
|---|---|
| `secao-01.md` | Abertura e orientacao |
| `secao-02.md` | Motivacao e problema |
| `secao-03.md` | Conceito minimo de Aho-Corasick |
| `secao-04.md` | Lacuna da literatura |
| `secao-05.md` | Estrategia adotada |
| `secao-06.md` | Metodologia experimental |
| `secao-07.md` | Resultados principais |
| `secao-08.md` | Diagnosticos de balanceamento |
| `secao-09.md` | Conclusao, limitacoes e futuro |
