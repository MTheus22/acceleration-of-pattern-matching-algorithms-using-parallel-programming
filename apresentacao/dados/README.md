# Dados intermediarios

Pasta para CSV/JSON extraidos para gerar graficos da apresentacao.

Nao tratar estes arquivos como fonte de verdade permanente. Eles sao derivados
do TCC ou do banco canonico de resultados e devem registrar:

- origem;
- data de geracao;
- comando ou script usado;
- secao do TCC usada para conferência.

Se um dado intermediario divergir do texto do TCC, pare e confira antes de usar.
