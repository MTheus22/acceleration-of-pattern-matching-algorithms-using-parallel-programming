# Roteiro

Pasta para o roteiro de fala da defesa.

O roteiro deve ser criado depois que `slides.md` estiver razoavelmente estavel.
Formato recomendado:

- um bloco por slide;
- objetivo do slide em uma frase;
- fala de 45 a 90 segundos;
- transicao para o proximo slide;
- observacoes para perguntas provaveis da banca.

O roteiro pode ter mais detalhe tecnico que o slide, mas deve continuar em
portugues claro e sem depender de termos internos do laboratorio.
