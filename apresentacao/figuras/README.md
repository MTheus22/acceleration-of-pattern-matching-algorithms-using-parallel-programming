# Figuras

Pasta para figuras usadas nos slides.

Regras:

- Nao copiar uma figura do TCC automaticamente. Primeiro verificar se ela fica
  legivel em 16:9.
- Preferir SVG/PDF vetorial para diagramas.
- Manter nomes descritivos, por exemplo:
  - `ac-sequencia-trie.svg`
  - `overlap-borda.svg`
  - `cache-footprint.svg`
  - `trafego-sharding-vs-texto.svg`
- Quando a figura vier de `../figuras/`, registrar isso no comentario do slide
  ou em nota no roteiro.

Figuras finais podem ficar diretamente aqui. Rascunhos ou fontes editaveis podem
ficar em subpastas criadas conforme necessario.
